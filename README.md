# EduWellProj
EduWellProj is a web-based educational system that empowers students to self-assess, monitor their progress, and check their grades in an organized and accessible way. It is designed to enhance the learning experience by promoting self-awareness, accountability, and academic improvement among learners.

Features
- Student Dashboard: View personal academic progress, grades, and performance summaries.
- Self-Assessment Tools: Students can evaluate their understanding and track their learning goals.
- Teacher Panel: Instructors can manage assessments, update grades, and monitor student performance.
- Course Management: Add, edit, or delete subjects and lessons.
- Secure Login System: User authentication for students, teachers, and admins.

- Database Integration: MySQL database ensures smooth data management and retrieval.
